2012年6月
发布于  2012/7/1 17:36:47
1.2012年6月12日，曾在本院作题为“共济共善：自治、多中心与发展”讲座的埃莉诺•奥斯特罗姆（Elinor Ostrom）教授逝世。奥斯特罗姆教授致力于公共资源管理的研究，并获得2009年诺贝尔经济学奖。作为政治学家、经济学家，她的研究证实了跨学科研究的重要性。 

2. 2012年6月18日上午，根据校学位评定委员会2012年第二次会议决议：2012年夏季，中欧法学院冯俊伟等2人获得法学博士学位、贾昂等31人获得法学硕士学位、时业伟等16人获得法律硕士学位。

3. 2012年6月20日下午，来自欧洲各国和欧盟机构的34名外交官来到本院，与学生展开题为“活力中国：民主与法治”的研讨。本次活动是中国外交部举办的“第七期欧洲外交官研讨班”活动的一部分，外交部欧洲司综合处副处长刘开源出席了座谈会。会议由本院2010级双硕士符琦琪主持。联席院长方流芳教授和Thomas Bruha教授分别致辞。

4. 2012年6月21日下午，本院第三届研究生分会主席团换届选举在明法楼306教室举行。2011级双硕士李冬同学当选为主席，杨国栋、夏彦当选为副主席。

5. 2012年6月27日上午，本院2012届毕业典礼在昌平校区明法楼305教室举行。中国政法大学校友、著名律师张思之先生、本院联席院长方流芳教授、布哈教授、教师代表以及本院100余名学生参加了毕业典礼。今年本院有47名2009级双硕士研究生、22名2011级欧洲-国际法学硕士研究生和2名2009级博士研究生毕业。

6. 2012年6月28日上午，中国政法大学2012届研究生毕业典礼暨硕士、博士学位授予仪式在学院路校区隆重举行。本院2012届博士毕业生冯俊伟代表毕业生作了题为《荣耀、使命与担当》的发言。
