中律原咨询（北京）有限公司招聘项目官员和学生助手
发布于  2009/12/7 8:36:19
中律原咨询(北京)有限公司成立于 2003年1月，是一家专门从事或协助从事与中国社会发展有关的非营利性项目的公司，主要领域涉及法制发展、政府治理、行政体制改革、管理技能、环境保护、正在兴起的非政府组织、教育、农村以及妇女问题等。公司现向中欧法学院学生提供就业和实习机会，具体事宜如下：

项目官员
1.  工作地点：北京

2.  职位描述：
（1） 开发、管理公司项目
（2） 协助公司合伙人和客户执行项目
（3） 协助高级职员就非营利性项目的设计、筹资、合规性等问题提供咨询意见
（4） 起草项目报告
（5） 进行项目评估
（6） 安排外国专家、客户来访

3.  要求
（1） 出色的中英文写作能力，普通话标准，英语口语流利
（2） 具有国内法学/法律硕士以上学位
（3） 有国外工作或留学经历
（4） 组织能力、沟通能力强
（5） 熟练操作电脑
（6） 对中国法制发展、非政府组织发展、法治进程和政府改革等问题有浓厚的兴趣，热心从事非营利项目

4.  待遇：根据应聘者资历和工作经验提供具有竞争力的薪酬及福利待遇。

5.  有意者请将英文申请信、中英文简历和中文习作（writing sample）通过电子邮件发送给张女士：zhonglvyuan@126.com

兼职学生助手
1. 职位描述：协助项目官员，负责项目的调研、书籍编辑出版、项目跟踪、日常联络协调等综合性工作；负责其他一些项目行政管理工作。

2. 工作类型：兼职，每周工作3个半天，其中2个半天在公司，能够工作1年以上者优先。

3. 待遇：面谈

4. 要求：
（1）对中国法制发展、公民权益保护、NGO发展和农村问题有浓厚的兴趣，热心从事非营利项目，关心社会问题
（2）在读法学硕士或者博士研究生
（3）出色的英文语言能力，能够阅读、回复英文的电子邮件，能够查找、阅读英文的一些资料
（4）熟练使用Word和Excel等办公软件

5.有意应聘的同学请将英文申请信及中英文简历通过电子邮件发送给张女士：zhonglvyuan@126.com ，来信注明中欧法学院学生。
