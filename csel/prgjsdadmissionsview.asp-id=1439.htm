2012年中国政法大学中欧法学院“法学博士”研究生招生简章（外国留学生专用）

1.简介

中国政法大学中欧法学院（以下简称：中欧法学院）2008年9月成立，是教育部批准的第一所中外合作办学的法学院。在中欧法学院成立之前，温家宝总理就对之寄予厚望，期待“中欧法学院能培养出大批学贯中西的法律人才”。

自成立以来，本院已招收2009、2010、2011三个年级，共计15名法学博士生，其中：中国学生13人，台湾学生1人，国际学生1人。

1.1 课程和师资
本院博士生课程以1978年之后中国市场化改革的历程为背景，从沿革、转型、现状和前瞻等层面展示中华人民共和国法制的基本框架和主要部分。同时，教授分别根据本人有所专攻的领域，就某一专门领域开设课程。本院专任教授来自中国政法大学，兼任教授来自北京、台北、美国等地的著名高校。

1.2 学习期限：3年

1.3 学位授予：中国政法大学法学博士学位

2.申请

2.1 申请人资格
申请人须为外国公民，须已取得（或到本院注册之前确定能取得）法学硕士学位或同等学位，具有一定汉语水平。

2.2 一份合格的申请应当包含以下全部文件
（1）护照复印件；
（2）中国法学博士项目申请表（点击下载）；
（3）研究计划（中文或英文）；
（4）发表物清单（列明文章或著作标题、发表时间、期刊或出版社）；
（5）申请人就读院校的教授推荐信：由教授本人将签字的推荐信扫描件发送至本院邮箱：admission_cesl@cupl.edu.cn；
（6）学位证复印件以及英文译本；
（7）与申请人法学教育经历有关的成绩单以及英文译本；
（8）汉语水平证明；
（9）申请人财务状况声明（点击下载）。

请将电子文本发送到学院邮箱：admission_cesl@cupl.edu.cn，需要签名的文件请扫描成PDF文档。申请人前来本院注册时，需提交与电子文本一致的、经申请人签名的纸质文件。

2.3 申请起止时间： 2011年12月1日至2012年4月1日

2.4 申请费用及缴纳方式
参见中国政法大学研究生院公布的《中国政法大学2012年招收外国来华留学生攻读博士、硕士学位招生简章》的相关规定：http://gate.cupl.edu.cn/eapdomain/ViewNote?nid=64284&pid=4942&ptid=2&unchecked=true，并照此执行。

3.学费、生活费和奖学金

3.1 学费：人民币33,000元/学年
除非得到本院事先同意，申请人应在开学登记时一次交纳当年学费。学生如在第一学年完成博士生培养方案所规定的课程学分，可仅缴纳一年学费。

3.2 除了学费之外，外国来华留学生尚有以下需要个人负担的开支：
（1）宿舍租金：2011年，本院留学生居住在中国政法大学昌平校区国际交流中心留学生宿舍，标准为两人一间，年租金10500元/床位，月租金1050元/床位，以上收费均包含水电费在内。2011年，在中国政法大学周边地区，租用一居室民房的租金为每月1500元左右。
（2）除租金之外，以2011年为例，一个学生用于食物、书籍、医疗保险、交通和交际等方面的日常生活费用，大约为每月2000元。

3.3 奖学金
3.3.1中国政法大学已列入国家留学基金委《接受中国政府奖学金来华留学生普通高等学校名录》，因此本院外国留学生可以向国家留学基金委申请中国政府奖学金，详见国家留学基金委网站：http://en.csc.edu.cn/Laihua/Default.aspx?cid=19。

3.3.2为培养具备原创性研究能力和国际视野的高层次法学研究人才，凡研究计划为本院认许，语言能力达到在欧洲进行学习和研究的要求，可获本院资助去欧洲合伙人院校从事为期一年的研究活动。

4．录取政策

（1）本院依赖申请人提供内容真实的申请材料，对申请人进行评估。择优选拔是本院基本的录取政策，本院鼓励申请人提供能够证明他（她）才能出众的材料。
（2）多元化的学生群体能够弘扬开放思维和包容精神，因此，本院致力于促成这样的学生群体。

本院保留修改和解释本章程的权利。
