“欧洲―国际法硕士”招生简章（中国大陆学生专用）
发布于  2011/3/15 10:36:28
1. “欧洲―国际法硕士”项目简介

1.1 课程和师资  
该项目的课程包括：9个单元必修课和7个单元选修课，每个单元由若干门相近或者相关课程构成。必修课单元包括：欧洲法律传统、法律写作与研究、国际公法、法治与人权、欧盟法律与政治简介、欧盟法（高级）、欧洲私法、政府和市场管制、WTO规则等主干课程。选修课单元包括：司法程序和争议解决、公共行政、比较刑法、国际商法、公司法、特殊经济领域法、自然资源和气候变化法等课程。只要选课人数满10人，就开设相应的选修课单元。

该项目的课程以欧洲法律传统、欧洲一体化和法律统一的趋势为背景，全面介绍欧盟法和国际法，而不是仅仅关注某一国家的某一部门法。同时，试图探索：欧洲国家如何在保持法律传统和法律融合之间保持平衡？

查看欧洲法课程一览表，请登陆：http://www.cesl.edu.cn/prgjmecurriculum.asp

本项目教授来自中欧法学院13所合伙人院校，它们是：德国汉堡大学、西班牙马德里自治大学、意大利博洛尼亚大学、法国斯特拉斯堡大学、法国巴黎政治学院、瑞典隆德大学、比利时鲁汶大学、爱尔兰都柏林大学圣三一学院、匈牙利中欧大学、匈牙利罗兰德依亚托斯大学、荷兰马斯特里赫特大学、波兰克拉科夫雅盖隆大学和英国曼彻斯特大学。通常是一个法学院的教授负责一个单元的课程。除来自合伙人院校的兼职教授之外，中欧法学院将聘用3―4名专任欧洲法教授。

1.2 学位论文 
从第二学期第9周开始，学生开始论文写作，每一名任课教授承担2-3名学生的论文指导。

1.3 教学语言：英语。

1.4 修学期限：一学年。每年9月开学，次年7月毕业。

1.5 学位授予：德国汉堡大学法学硕士学位。

2. 申请
2.1 申请人资格 
申请本项目需符合以下条件：
（1）须为中国大陆居民；
（2）须已获得（或到本院注册之前定能获得）法学学位，或者已通过国家司法考试。

2.2 申请起始时间：2011年3月15日-2011年5月25日。

2.3  一份合格的申请应当包含的以下全部文件
（1）身份证复印件
（2）申请人亲自填写的项目申请表（点击下载）
（3）申请人曾就读院校的教授的推荐信：由教授本人将签字的推荐信扫描件发送到本院电子邮箱：admission_cesl@cupl.edu.cn 或 christine17chen@hotmail.com
（4）学位证、毕业证复印件
（5）与申请人法学教育经历有关的成绩单
（6）申请人财务状况声明（点击下载）
（7）申请英语免试的，还须提交TOEFL或IELTS考试成绩单复印件，或者以英语完成本科教育的证明复印件。

本院鼓励学生提交电子文本申请材料至：admission_cesl@cupl.edu.cn 或 christine17chen@hotmail.com
，学生前来本院注册时，需提交与电子文本一致的、经本人签名的纸质文件。

3. 学费、生活费和奖学金
3.1 学费：人民币30,000元/学年。

除非得到本院事先同意，申请人应在入学登记时一次交纳全部学费。申请缓交学费的，应在“财务状况声明”中载明缓交数额、延期支付时间。

3.2 奖学金
本院为优胜学生提供奖学金。新生入学奖学金在学生注册之后的三周内提供。

4．录取政策
（1）本院依赖申请人提供内容真实的申请材料，对申请人进行评估。择优选拔是本院基本的录取政策，本院鼓励申请人提供能够证明他（她）才能出众的材料。
（2）申请人无须参加全国硕士研究生入学考试。
（3）申请人需参加本院组织的入学考试，包括：英语听力、法律分析（英文作答）和面试，考试成绩权重分别为：英语听力占20%，法律分析占50%，面试占30%。中欧法学院结合考生的申请材料和入学考试成绩进行综合评估，决定是否录取。
（4）多元化的学生群体能够弘扬开放思维和包容精神，因此，本院致力于促成这样的学生群体。

本院保留修改和解释本章程的权利。
