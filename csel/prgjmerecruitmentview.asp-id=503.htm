“欧洲―国际法硕士”招生简章（2010级外国留学生专用）
发布于  2010/2/25 10:36:32
1. “欧洲―国际法硕士”项目简介

1.1  课程和师资  
本项目由9个单元必修课程和一系列选修课单元组成，每个单元又由若干课程构成。必修单元包括：欧洲法律传统、法律写作与研究、国际公法、法治与人权、欧盟法律与政治简介、欧盟法（高级）、欧洲私法、政府和市场管制、WTO规则等主干课程。选修单元提供司法程序和争议解决、公共行政、国际贸易、公司法、中国公法与私法等课程。10人为选修单元开课的人数下限。

本课程以欧洲法律传统、欧洲一体化和法律统一的趋势为背景，全面介绍欧盟法和国际法，而不是仅仅关注某一国家的某一部门法。与此同时，本课程旨在探索：欧洲国家如何在保持法律传统和法律融合之间保持平衡。

查看欧洲法课程一览表，请登陆：http://www.cesl.edu.cn/prgjmcourse.asp

本项目教授来自中欧法学院13所合伙人院校，它们是：德国汉堡大学、西班牙马德里自治大学、意大利博洛尼亚大学、法国斯特拉斯堡大学、法国巴黎政治学院、瑞典隆德大学、比利时鲁汶大学、爱尔兰都柏林大学圣三一学院、匈牙利中欧大学、匈牙利罗兰德依亚托斯大学、荷兰马斯特里赫特大学、波兰克拉科夫雅盖隆大学和英国曼彻斯特大学。通常是一个法学院的教授负责一个单元的课程。除来自合伙人院校的兼职教授之外，中欧法学院将聘用3-4名专任欧洲法教授，中国法课程将由中国知名院校的学者讲授。

1.2 学位论文 
从第二学期第9周开始，学生开始论文写作，每一名任课教授承担2-3名学生的论文指导。

1.3 教学语言：英语。

1.4 修学期限：一学年。每年9月开学，次年7月毕业。

1.5 学位授予：德国汉堡大学法学硕士学位。

2. 申请
2.1 申请人资格 
申请本项目需符合以下条件：
（1）须为外国公民；
（2）须取得（或到本院注册之前确定能取得）所在国的第一法学学位（包括学士学位、通过国家司法考试或其他同等学位）；
（3）须有足以完成欧洲法学习的英语水平，并提交以下证明文件之一：
――以英语为母语或以英语完成本科教育；
――或者有效期内的、不低于100分的托福（TOEFL）成绩，或者有效期内的、不低于7分的雅思（IELTS）成绩；

2.2 申请起始时间
每年11月20日至次年4月20日。

2.3 一份合格的申请应当包含的以下全部文件
（1）护照复印件
（2）一寸数码照片
（3）申请人亲自填写的“欧洲―国际法” 法学硕士项目申请表（选择一：在线申请/选择二：点击下载）
（4）申请人曾就读院校的教授的推荐信：由教授本人将签字的推荐信扫描件发送到本院电子邮箱：admission_cesl@cupl.edu.cn
（5）学位证复印件
（6）与申请人法学教育经历有关的成绩单
（7）TOEFL或IELTS考试成绩单复印件，或者以英语完成本科教育的证明复印件
（8）申请人财务状况声明（点击下载）

本院鼓励学生提交电子文本申请材料至：admission_cesl@cupl.edu.cn，学生前来本院注册时，需提交与电子文本一致的、经本人签名的纸质文件。

3. 学费、生活费和奖学金

3.1 学费：人民币60,000元/学年。

除非得到本院事先同意，申请人应在入学登记时一次交纳全部学费。申请缓交学费的，应在“财务状况声明”中载明缓交数额、延期支付时间。

3.2 除了学费之外，外国留学生尚有以下需要个人负担的开支：
（1）宿舍租金
2009年，外国留学生居住在中国政法大学昌平校区国际交流中心留学生宿舍，标准为两人一间，年租金人民币10500元/一床位,月租金人民币1050元/一床位,以上收费均包含水电费在内。
2009年，在中国政法大学周边地区，租用一居室民房的租金为每月1500元左右。

（2）除租金之外，以2009年为例，一个外国留学生用于食物、书籍、医疗保险、交通和交际等方面的日常生活费用，大约为每月2000元。

3.3 奖学金
本院、中国政法大学分别设有资助外国留学生的奖学金。

3.3.1本院为优胜学生提供奖学金。

新生入学奖学金在学生注册之后的三周内提供。

3.3.2外国留学生也可以申请中国政法大学设立的奖学金（每月最高额2000元，最长资助期限一年），具体步骤如下：
（1）申请人向中欧法学院提交申请；
（2）中国政法大学国际合作与交流处决定是否给予奖学金。

4．录取政策
（1）本院依赖申请人提供内容真实的申请材料，对申请人进行评估。对于外国留学生，本院不设入学考试，但是，不排除通过电话通话、委托校友面谈等方式，对申请人进行评估。
（2）择优选拔是本院基本的录取政策，本院鼓励申请人提供能够证明他（她）才能出众的材料。
（3）国籍、文化和种族多元的学生群体能够弘扬开放思维和包容精神，因此，本院致力于促成这样的学生群体。
