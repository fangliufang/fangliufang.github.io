“欧洲-国际法硕士”项目招生简章（2013级外国留学生专用）
发布于  2013/2/5 11:00:45
1. “欧洲-国际法硕士”项目简介

1.1  课程和师资 

该项目的课程包括：9个单元必修课和5个单元选修课，每个单元由若干门相近或者相关课程构成。必修课单元包括：欧洲法律传统、法律写作与研究、国际公法、法治与人权、人权和法治、欧盟法律与政治简介、欧盟法（高级）、欧洲私法、国际经济法、法律管制等主干课程。选修课单元包括：司法程序和争议解决、国际商法、公司法等课程。只要选课人数满10人，就开设相应的选修课单元。

本课程以欧洲法律传统、欧洲一体化和法律统一的趋势为背景，全面介绍欧盟法和国际法，而不是仅仅关注某一国家的某一部门法。与此同时，本课程旨在探索：欧洲国家如何在保持法律传统和法律融合之间保持平衡。

查看欧洲法课程一览表，请登陆：http://www.cesl.edu.cn/prgjmdcourse.asp 

本项目教授来自中欧法学院13所合伙人院校，它们是：德国汉堡大学、西班牙马德里自治大学、意大利博洛尼亚大学、法国斯特拉斯堡大学、法国巴黎政治学院、瑞典隆德大学、比利时鲁汶大学、爱尔兰都柏林大学圣三一学院、匈牙利中欧大学、匈牙利罗兰德依亚托斯大学、荷兰马斯特里赫特大学、波兰克拉科夫雅盖隆大学和英国曼彻斯特大学。通常是一个法学院的教授负责一个单元的课程。

1.2 学位论文

从第二学期第9周开始，学生开始论文写作，每一名任课教授承担2-3名学生的论文指导。

1.3 教学语言：英语。

1.4 修学期限：一学年。

1.5 学位授予：德国汉堡大学法学硕士学位。

2. 申请

2.1 申请人资格 

申请本项目需符合以下条件：
（1）须为外国公民；
（2）须取得（或到本院注册之前确定能取得）所在国的第一法学学位（包括学士学位、通过国家司法考试或其他同等学位）；
（3）须有足以完成欧洲法学习的英语水平，并提交以下证明文件之一：
     ――以英语为母语或以英语完成本科教育；
     ――或者有效期内的、不低于100分的托福（TOEFL）成绩，或者有效期内的、不低于7分的雅思（IELTS）成绩；

2.2 申请时间：2013年2月5日-2013年5月1日。

2.3 一份合格的申请应当包含的以下全部文件

（1）护照复印件
（2）申请人亲自填写的“欧洲-国际法学硕士”项目申请表（点击下载）
（3）申请人曾就读院校的教授的推荐信（英文）：由教授本人将签字的推荐信扫描件发送到本院电子邮箱：meil_cesl@cupl.edu.cn
（4）学位证复印件及其英文译本
（5）与申请人法学教育经历有关的成绩单、评分标准及其英文译本
（6）TOEFL或IELTS考试成绩单复印件，或者以英语完成本科教育的证明复印件
（7）申请人财务状况声明（点击下载）

本院鼓励学生提交电子文本申请材料至：meil_cesl@cupl.edu.cn，学生前来本院注册时，需提交与电子文本一致的、经本人签名的纸质文件。

3. 学费、生活费和奖学金

3.1 学费：人民币60,000元/学年。
除非得到本院事先同意，申请人应在入学登记时一次交纳全部学费。申请缓交学费的，应在“财务状况声明”中载明缓交数额、延期支付时间。

3.2 除了学费之外，外国留学生尚有以下需要个人负担的开支：

（1）宿舍租金
外国留学生居住在中国政法大学昌平校区国际交流中心留学生宿舍，标准为两人一间，年租金人民币10500元/床位,上述收费包含水电费在内。

2012年，在中国政法大学周边地区，租用一居室民房的租金为每月2000元左右。

（2）除租金之外，以2012年为例，一个外国留学生用于食物、书籍、医疗保险、交通和交际等方面的日常生活费用，大约为每月2000元。

3.3 奖学金

3.3.1本院为优胜学生提供奖学金。新生入学奖学金在学生注册之后的三周内提供。

3.3.2中国政法大学已列入国家留学基金委《接受中国政府奖学金来华留学生普通高等学校名录》，因此申请本院硕士项目的外国留学生可以向国家留学基金委申请中国政府奖学金，申请期为每年1月至4月，详见国家留学基金委网站：http://en.csc.edu.cn/Laihua/Default.aspx?cid=19

4．录取政策

（1）本院依赖申请人提供内容真实的申请材料，对申请人进行评估。对于外国留学生，本院不设入学考试，但是，不排除通过电话通话、委托校友面谈等方式，对申请人进行评估。
（2）择优选拔是本院基本的录取政策，本院鼓励申请人提供能够证明他（她）才能出众的材料。
（3）国籍、文化和种族多元的学生群体能够弘扬开放思维和包容精神，因此，本院致力于促成这样的学生群体。
